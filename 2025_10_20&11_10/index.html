Meine Recherche zu HTML

Elemente

HTML-Elemente sind die Bausteine einer Webseite.
Die interessantesten Elemente meiner Meinung nach sind:

  • line break (br)
    Es erzeugt einen Zeilenumbruch und ist ein leeres (self-closing)
    Das bedeutet, es hat keinen Inhalt und keinen schließenden Tag.
  • list item (li)
    Es macht einen Listenpunkt, wird innerhalb von (ul) (ungeordnete Liste) oder (ol) (geordnete Liste) verwendet.
  • unordered list (ul)
    Liste ohne Reihenfolge, die Punkte werden als Aufzählungszeichen (•) angezeigt.
  • ordered list (ol)
    Liste mit einer bestimmten Reihenfolge, die Punkte werden nummeriert (1., 2., 3., ...).
Es gibt noch viele weitere Elemente, wie html, body, paragraph usw.
Ich habe jetzt die mal ausgwählt, weil ich sie im schreiben von diesen Text benützt habe.
  • tabelle (table)
    Erstellt eine Tabelle, die aus Zeilen (tr) und Zellen (td) besteht.

    Obst Menge
    Apfel 25
    Birne 2
  • form / input
    Erstellt ein Formular mit Eingabefeldern.

    Name:

    E-Mail:

    form umschließt die Eingabefelder
    input type="text" für normalen Text
    input type="email" für E-Mail
    input type="submit" zum Absenden

Attribute

HTML-Attribute sind zusätzliche Informationen, die man einem HTML-Element mitgibt, um sein Verhalten, Aussehen oder seine Funktion zu verändern.
Die interessantesten Attribute meiner Meinung nach sind:

  • Hypertext Reference (href)
    Gehört zum (a)-Element (Link)
    Mit dem href-Attribut wird die URL der Seite angegeben, zu der der Link führt.
    Beispiel:
    Besuche W3Schools
  • src (Source)
    Gibt die Quelle einer externen Ressource an, wie z.B. ein Bild oder ein Skript. Es wird im (img) oder (script) Element verwendet.

  • alt (Alternative Text)
    Bietet eine Textbeschreibung für ein Bild, die angezeigt wird, wenn das Bild nicht geladen werden kann. Es wird im (img) Element verwendet.

  • class
    Weist einem HTML-Element eine oder mehrere Klassen zu, die für CSS-Styling oder JavaScript verwendet werden können.
Es gibt noch viele weitere Attribute, wie id, style, title usw.
